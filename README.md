## 🤝 共同建设这个知识库

这个仓库追求**有分量的内容**，而非廉价的数量。如果你：

1. ✏️ **有 Web3 实战经验** — 写一篇带心得的教程
2. 📊 **做过深度链上分析** — 分享你的方法论和看板
3. 📝 **写过高质量的项目研报** — 上传到投研中心
4. 🔧 **亲自验证过好用的工具** — 推荐给社区并附上使用体验
