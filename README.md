## 🤝 共同建设

这个仓库追求**有分量的内容**，而非廉价的数量。如果你有 Web3 实战经验或深度见解：

- ✏️ **有 Web3 实战经验** → 写一篇带心得的教程
- 📊 **做过链上分析** → 分享你的方法论和看板
- 📝 **写过高质量研报** → 上传到投研中心
- 🔧 **亲自验证过好用的工具** → 附上使用体验推荐
