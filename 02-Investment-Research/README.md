## 研报框架

一份完整的项目研报应该覆盖以下 8 个维度：

```
1️⃣ 项目概要 → 一句话定位 + 核心亮点
2️⃣ 赛道分析 → 市场空间、竞争格局
3️⃣ 团队与融资 → 背景资方、估值逻辑
4️⃣ 产品与技术 → 架构、创新点、安全审计
5️⃣ 代币经济 → 分配、解锁、价值捕获
6️⃣ 数据验证 → TVL / 用户 / 收入真实数据
7️⃣ 风险提示 → 技术 / 监管 / 竞争风险
8️⃣ 综合评分 → 投资评级
```
