# Web3 基础概念

## 什么是 Web3？

Web3 是互联网的第三代范式——**去中心化、用户拥有数据主权、价值可以在网络中原生流转**。

| 对比 | Web1 (只读) | Web2 (读写) | Web3 (读写+拥有) |
|------|------------|------------|----------------|
| 时代 | 1990-2005 | 2005-2020 | 2020+ |
| 特征 | 静态网页 | 社交平台 | 去中心化应用 |
| 数据 | 服务器存储 | 平台拥有 | 用户掌控 |
| 代表 | Yahoo, Netscape | Facebook, Twitter | Ethereum, Solana |

## 区块链核心概念

### 区块 (Block)
数据打包单元，包含交易记录和时间戳。

### 链 (Chain)
区块按时间顺序链接，不可篡改。

### 共识机制
- **PoW** (工作量证明)：比特币，算力竞赛
- **PoS** (权益证明)：以太坊2.0，质押验证
- **DPoS** (委托权益证明)：EOS, TRON

### 智能合约 (Smart Contract)
链上自动执行的代码逻辑——"Code is Law"。

## 关键术语速查

| 术语 | 解释 |
|------|------|
| **Gas Fee** | 交易手续费，按计算量计价 |
| **Nonce** | 账户交易序号，防重放攻击 |
| **Slippage** | 滑点，交易执行价与预期价的差异 |
| **TVL** | Total Value Locked，锁仓总价值 |
| **APR/APY** | 年化利率/年化收益率 |
| **Liquidity** | 流动性，市场深度 |
| **Impermanent Loss** | 无常损失，LP 提供者的风险 |
| **MEV** | 矿工可提取价值，交易排序套利 |

> 💡 **建议：** 不用一次性记住所有术语，在实际操作中边用边学
