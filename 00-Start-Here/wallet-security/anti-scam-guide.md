# 🔐 钱包安全与防骗指南 (必读)

> ⚠️ 在 Web3 世界，你的安全 = 你的习惯。没有银行客服可以帮你找回资产。

## 一、钱包类型选择

| 类型 | 推荐 | 适用场景 | 安全等级 |
|------|------|---------|---------|
| **硬件钱包** | Ledger, Trezor, OneKey | 大额资产存储 (>$1000) | ⭐⭐⭐⭐⭐ |
| **浏览器插件** | Rabby, MetaMask, OKX Wallet | 日常交互、小额操作 | ⭐⭐⭐ |
| **移动端** | Rainbow, MetaMask Mobile | 小额、临时使用 | ⭐⭐⭐ |
| **交易所** | Binance, OKX | 出入金、交易高频 | ⭐⭐ (平台风险) |

> 🏆 **推荐组合：** Ledger (金库) + Rabby (日常) + 交易所 (交易)

## 二、助记词安全铁律

### ✅ 正确做法
- 手写助记词在**防火防水**的纸上，存两处不同地点
- 用钢片压印 (如 CryptoSteel)，防火防洪水
- 硬件钱包生成助记词，永不接触网络设备

### ❌ 绝对禁止
- ❌ 截图 / 手机拍照 → iCloud/Google Photos 自动同步即泄露
- ❌ 存在微信收藏 / 笔记软件 / 网盘
- ❌ 在任何网页输入助记词 (100% 是钓鱼)
- ❌ 通过任何通讯软件发送 (包括给"客服")

## 三、日常操作安全 Checklist

```
每次交互前过一遍 ✅
□ 检查 URL 是否拼写正确 (uniswap.com ≠ uniswaap.com)
□ 检查合约地址是否官方 (从 CoinGecko/DefiLlama 确认)
□ 不要看到 Approve 就点——看授权金额
□ 小额定金先试试，确认没问题再大额操作
□ 撤销不需要的授权 (用 revoke.cash)
```

## 四、常见骗局识别

### 🎣 钓鱼攻击
伪装成知名网站的假链接，诱导你连接钱包或输入私钥。
> **自检方法：** 收藏真实官网，每次都从收藏夹打开

### 🏃 Rug Pull
项目方制造热度后，突然撤走流动性跑路。
> **红线指标：** 🔴 匿名团队 | 🔴 锁仓期过短 | 🔴 代币集中度高

### 💀 Approve 钓鱼
恶意合约获得你代币的无限授权额度，可以随时转走。
> **防御：** 用 Rabby 钱包 (自动识别风险授权)，用完及时 Revoke

### 🤖 假客服 / 假社群
Discord/Telegram 里冒充管理员私信你"帮忙解决问题"。
> **记住：** 真正的管理员永远不会主动 DM 你。

## 🆘 如果已经被骗了

1. **不要慌，停止所有链上操作**
2. 如果只是授权被钓鱼 → 立刻撤销授权
3. 如果是私钥泄露 → 马上创建新钱包，把剩余资产转走
4. 在社群发布警告，防止更多人受害
5. **承认损失，吸取教训**——这是 Web3 最贵的一课
