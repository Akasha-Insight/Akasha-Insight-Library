## 🙋 常见问题

**Q: 我需要买币才能开始学习吗？**
A: 不需要！先用测试网 (Testnet) 和公开链上数据免费学习。

**Q: 没有编程基础能学链上分析吗？**
A: 完全可以。从 Dune 的可视化看板开始，SQL 可以边用边学。
