# 📊 链上数据分析 (On-Chain Analysis)

> 用数据读懂链上世界——从 SQL 到看板，从工具到策略

## 本模块内容

| 目录 | 说明 |
|------|------|
| 🗄️ [dune-analytics](./dune-analytics/) | Dune SQL 从入门到进阶，实战看板 |
| 🕵️ [nansen-arkham](./nansen-arkham/) | Nansen/Arkham 数据分析技巧 |
| 🛠️ [tools](./tools/) | DeBank, DefiLlama, DexTools 等工具教程 |

## 为什么要学链上分析？

```
传统金融 → 财报延迟、数据不透明
链上数据 → 实时、公开、不可篡改
```

🎯 **你可以追踪：**
- Smart Money（聪明钱）的动态
- 协议 TVL 的变化趋势
- 巨鲸的建仓/出货行为
- 新代币的早期发现信号

## 推荐学习路径

```
1️⃣ 先看 Dune 公开看板 → 感受数据的可视化力量
2️⃣ 学基础 SQL → 写自己的查询
3️⃣ 用 Nansen/Arkham → 标签化追踪地址
4️⃣ 组合工具 → 构建自己的分析体系
```
